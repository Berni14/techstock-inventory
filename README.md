# techstock-inventory
Sistema de gestión de inventario en Java para tiendas tecnológicas. Implementa CRUD, persistencia en ficheros y lógica de alertas de stock.

Tecnologias usadas:
- Java 17+
- Programación Orientada a Objetos (POO)
- Serialización de Objetos (Persistencia en ficheros .dat)
- Java Streams & Lambda Expressions
